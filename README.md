# Audit Tracker — ทะเบียนงบสอบบัญชี

เว็บแอปติดตามงบสอบบัญชี (แทนไฟล์ Excel) — Dashboard, รายการงบ, ติดตามเอกสาร/เก็บเงิน, Import/Export Excel

- ข้อมูลผู้ใช้เก็บใน localStorage ของเครื่องแต่ละเครื่อง
- ข้อมูลตั้งต้นใน `seed-data.enc.js` ถูกเข้ารหัส AES-256-GCM (PBKDF2 250k) — ต้องมีรหัสปลดล็อคจึงอ่านได้
